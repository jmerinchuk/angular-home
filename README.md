#Overview
This project is built with AngularJS v1.6.9, HTML, CSS, and JavaScript. 
The goal is to practice core Angular concepts by gradually developing a simple application that includes a homepage, navigation between multiple views, and basic HTTP service calls.

#Project Progress
The repository shows the evolution of the app from a minimal starter structure with a few controllers into a more complete idea. 
Each stage reflects incremental learning and experimentation with AngularJS patterns, routing, and component organization.
